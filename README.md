## custom Pubterminal settings

Displays the stuff below in terminal everytime you load up the terminal or sign into ssh
* hostname
* cpu
* address
* Uptime
* memory
* username
* privlages
* sessions
* processes
* packages

## Dependencies
Your gonna need figlet package and a PxPlus IBM VGA 10 font installed
